== Overview

Implement the message templates feature for BRF inspection schedules. The feature allows administrators to create, edit, preview, and manage message templates with user-defined placeholders that map to building and schedule data fields. 3 phases, 9 tasks.

== Phase 1: Foundation

Data model and template engine -- pure TypeScript functions with no UI.

* [x] TASK-001: Create template data model interfaces [parallel]
+
Components:: template.ts
Stories:: US-001, US-002, US-005
Specs:: TMPL-002, TMPL-004, TMPL-005, TMPL-007, TMPL-020
Description:: Create `src/lib/models/template.ts` with `PlaceholderMapping`, `DataField`, and `MessageTemplate` interfaces as specified in the design.

* [x] TASK-002: Implement template engine business logic [sequential]
+
Components:: template-model.ts
Stories:: US-001, US-002, US-004, US-005
