== Overview

Implementation of the building configuration feature -- the first feature in the project. Includes project scaffolding (Vite 6 + Svelte 5 + TypeScript + Vitest), data model, persistence layer, business logic, and UI components with SVG facade illustration. Phases 1-4 are complete (initial implementation). Phases 5-6 add uniform apartment count mode and enhanced SVG facade. 6 phases, 16 tasks.
