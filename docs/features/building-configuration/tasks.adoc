== Phase 9: Flexible Apartment Digit Assignment

Specs:: BCFG-048, BCFG-049, BCFG-052

* [x] TASK-027: Update validation functions for digit positions and expanded range [parallel]
+
Components:: src/lib/models/building-model.ts

Ensure unique IDs by the (floor, position) → ID mapping.
